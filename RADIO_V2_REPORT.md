# Отчёт: страница «Радио» на базе дизайн-системы v2

## Реализация

- **hero hero--radio** — фон, overlay, title, subtitle, actions (2 кнопки)
- **benefits benefits--radio** — 4 карточки (Импульс, Позиционирование, Локализация, Наш подход)
- **media-split media-split--poster media-split--radio** — изображение робота, 3 карточки (Призыв по номеру, Голосовой AI, Теплая заявка)
- **steps steps--radio** — 5 карточек (Объективность, Точный таргетинг, AI-усиление, Full-service, Оптимизация бюджета)
- **feature-grid feature-grid--radio** — 4 карточки (Импульсные акции, Построение бренда, Продвижение услуг, Массовый найм)
- **cta-form cta-form--radio** — заголовок, текст, форма (4 поля: имя, телефон, компания, интересующий формат; кнопка)
