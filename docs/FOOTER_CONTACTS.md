# Как работают контакты в подвале и в блоке «Свяжитесь с нами»

## Три иконки в подвале (слева под логотипом)

| Иконка   | Куда ведёт ссылка | Откуда берётся значение |
|----------|-------------------|--------------------------|
| **Telegram** | Чат с вашим номером в Telegram (`https://t.me/+79179816000`) | Номер из **CONTACT_PHONE** (.env или по умолчанию `79179816000`). Ссылка всегда собирается так: `https://t.me/+` + цифры номера. |
| **Макс**     | Чат в мессенджере Макс по вашему номеру | Ссылка из **CONTACT_MAX_URL** (.env). Если не задана — иконка ведёт на `#`. Ссылку нужно скопировать в приложении Макс (профиль → поделиться). |
| **Почта**    | Открыть почтовый клиент и написать на ваш email | Адрес из **CONTACT_EMAIL** (.env или по умолчанию `welcome@ad-smart.ru`). В шаблоне подставляется как `mailto:{{ CONTACT_EMAIL }}`. |

## Откуда код берёт значения

1. **Настройки** (`adsmart/settings.py`):
   - `CONTACT_PHONE` — номер для Telegram и для блока «Свяжитесь с нами» (только цифры, без `+`).
   - `CONTACT_PHONE_DISPLAY` — как показывать номер пользователю (например `+7 (917) 981-60-00`).
   - `CONTACT_EMAIL` — email для связи.
   - `CONTACT_MAX_URL` — готовая ссылка на чат в Максе (из приложения).

2. **Контекст** (`core/context_processors/project.py`):
   - Читает эти настройки и отдаёт в шаблоны:
     - `FOOTER_TELEGRAM_URL` = `https://t.me/+{CONTACT_PHONE}` (цифры из `CONTACT_PHONE`).
     - `FOOTER_MAX_URL` = `CONTACT_MAX_URL` или `#`.
     - `FOOTER_EMAIL` = `CONTACT_EMAIL`.
   - Там же формируются `CONTACT_PHONE`, `CONTACT_PHONE_DISPLAY`, `CONTACT_EMAIL` для блока «Свяжитесь с нами».

3. **Шаблон подвала** (`templates/components/demo/footer.html`):
   - Иконка Telegram: `href="{{ FOOTER_TELEGRAM_URL }}"` — подставляется значение из контекста.
   - Иконка Макс: `href="{{ FOOTER_MAX_URL }}"`.
   - Иконка почты: `href="mailto:{{ FOOTER_EMAIL }}"`.

## Почему может открываться «страница Telegram»

- Ссылка `https://t.me/+79179816000` открывает **чат с этим номером**. В браузере это часто показывается как страница на домене Telegram (например web.telegram.org или t.me) — это нормально, это и есть страница чата.
- Если открывается главная telegram.org без чата — тогда подставляется не тот URL. Проверьте:
  - В .env **нет** переменной `FOOTER_TELEGRAM_URL` (мы её больше не используем).
  - В .env можно задать только: `CONTACT_PHONE=79179816000` (цифры без пробелов и плюса).
- После смены .env перезапустите сервер Django и обновите страницу с полным сбросом кэша (Ctrl+F5).

## Что задать в .env (по желанию)

```env
CONTACT_PHONE=79179816000
CONTACT_PHONE_DISPLAY=+7 (917) 981-60-00
CONTACT_EMAIL=welcome@ad-smart.ru
CONTACT_MAX_URL=   # сюда вставить ссылку на чат из приложения Макс
```

Если не задавать — подставятся значения по умолчанию из `settings.py`.
