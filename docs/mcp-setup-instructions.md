# Инструкция по настройке MCP сервера Figma в Cursor

## Шаг 1: Открыть настройки Cursor

1. Откройте Cursor
2. Нажмите `Ctrl+,` (или `Cmd+,` на Mac) для открытия настроек
3. Или через меню: `File` → `Preferences` → `Settings`

## Шаг 2: Найти раздел MCP

В поиске настроек введите: **"MCP"** или **"Model Context Protocol"**

## Шаг 3: Добавить конфигурацию Figma MCP

Найдите поле для конфигурации MCP серверов и добавьте:

```json
{
  "mcpServers": {
    "FigmaRemoteMcp": {
      "url": "http://mcp.figma.com/mcp"
    }
  }
}
```

**Или** если настройки в формате JSON (через `settings.json`):

1. Нажмите на иконку `{}` в правом верхнем углу настроек (открыть JSON)
2. Добавьте конфигурацию в корень файла:

```json
{
  "mcpServers": {
    "FigmaRemoteMcp": {
      "url": "http://mcp.figma.com/mcp"
    }
  },
  // ... остальные настройки
}
```

## Шаг 4: Сохранить и перезапустить

1. Сохраните настройки (`Ctrl+S` или `Cmd+S`)
2. **Обязательно перезапустите Cursor** для применения изменений

## Шаг 5: Проверка подключения

После перезапуска Cursor, проверьте подключение:

1. Откройте чат с AI ассистентом
2. Попросите проверить MCP ресурсы: "Проверь доступность MCP ресурсов Figma"
3. Или попробуйте получить данные из Figma напрямую

## Альтернативный способ (через конфигурационный файл)

Если настройки через UI не работают, можно попробовать добавить конфигурацию в файл:

**Windows:**
- `%APPDATA%\Cursor\User\settings.json`

**macOS:**
- `~/Library/Application Support/Cursor/User/settings.json`

**Linux:**
- `~/.config/Cursor/User/settings.json`

## Что делать, если не работает

1. **Проверьте URL**: Убедитесь, что URL правильный: `http://mcp.figma.com/mcp`
2. **Проверьте версию Cursor**: MCP может требовать определенную версию
3. **Проверьте логи**: В Cursor могут быть логи ошибок подключения
4. **Используйте альтернативу**: Можно работать с существующим `design/responsive-home.json`

## После успешной настройки

Когда MCP будет настроен, вы сможете:
- Получать данные напрямую из Figma
- Экспортировать актуальные данные для 834-1439px
- Обновлять JSON файлы автоматически
