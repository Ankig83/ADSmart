# Настройка MCP сервера для Figma

## Конфигурация MCP сервера

Для работы с Figma через MCP сервер необходимо добавить следующую конфигурацию в настройки Cursor:

### Где настроить:
1. Откройте настройки Cursor: `File` → `Preferences` → `Settings` (или `Ctrl+,`)
2. Найдите раздел "Model Context Protocol" или "MCP Servers"
3. Добавьте следующую конфигурацию:

```json
{
	"mcpServers": {
		"FigmaRemoteMcp": {
			"url": "http://mcp.figma.com/mcp"
		}
	}
}
```

### Альтернативный способ (через конфигурационный файл):
Если Cursor поддерживает конфигурацию через файл, создайте файл `.cursor/mcp.json` в корне проекта:

```json
{
	"mcpServers": {
		"FigmaRemoteMcp": {
			"url": "http://mcp.figma.com/mcp"
		}
	}
}
```

## Проверка подключения

После настройки MCP сервера:
1. Перезапустите Cursor
2. Проверьте доступность ресурсов через команду `list_mcp_resources`
3. Если ресурсы доступны, можно использовать MCP для получения данных из Figma

## Использование

После настройки MCP можно:
- Получать данные напрямую из Figma через MCP API
- Экспортировать актуальные данные для разрешения 836-1339px
- Обновлять существующие JSON файлы

## Примечание

Если MCP сервер недоступен, можно использовать:
- Существующий файл `design/responsive-home.json`
- Скрипт `scripts/figma_export.py` для экспорта данных через Figma API
